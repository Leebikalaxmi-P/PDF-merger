PDF Merger Using PyPDF2

A simple and efficient Python application that merges multiple PDF files into a single PDF document using the PyPDF2 library. This project demonstrates the practical use of Python for document automation and file handling.

📖 Project Description

Managing multiple PDF files can be inconvenient when documents need to be shared, printed, or archived together. This project provides a straightforward solution by allowing users to combine multiple PDF files into one consolidated document.

The application uses the PyPDF2 library to read and merge PDF files while preserving the original content and page order.

🔍 How It Works
The program imports the PdfMerger class from PyPDF2.
A list of PDF files is created.
Each PDF is added to the merger object.
The merged content is written to a new PDF file.
Resources are released using the close() method.
